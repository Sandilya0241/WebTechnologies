Basics of JavaScript

JavaScript code can be placed between
<script>
...
</script> tags of HTML file. This we call it inline JavaScript. This isn't actually a best practice as we don't have demarcation between our markup and JavaScript code.

Why do we need to have demacation?
  • One in case of multiple developers are working together.
  • Second in case of reusability.
How to separate this JavaScript from Markup language?

With below syntax:
<script src="/filename.js"></script>

What is a Variable?

Data that is stored in Computer memory to hold information has a random address number. To access that memory location for the information, we set a name to that location called Variable.
Example:

  • let total = 149.99;
  • let product=Hiking Boots;
  • let discounted = true;

When variable is uninitialized, it will have a value 'undefined'

What is a Constant?

This is never going to change. Once set in the beginning, it cannot be set to a new value.

Example:
const constantName = "value";
constantName = "new value"; (this would throw a type error saying we are initializing to a const)
